overview
the idea.

most run-trackers turn the hobby into a leaderboard. runera goes the other way — a calm, encouraging companion that rewards consistency over speed.

each run logged builds a streak; each streak unlocks unique on‑chain collectibles minted to the runner's wallet, claimable directly from the dashboard.

auth is email‑and‑google through privy, so non‑crypto users never see a seed phrase. the wallet is provisioned silently in the background.

what i built

the system, end to end.

01

frictionless onboarding

email or google sign-in via privy. a wallet is created silently behind the scenes so the user never has to think about keys, gas, or chains.

privy oauth
02

run logging & streaks

users log runs by distance and pace. a backend job recomputes streak state nightly and pushes streak‑broken / streak‑grew events back to the client.

postgres cron
03

collectible rewards

each streak tier maps to a unique nft. on tier-up the contract mints to the user's embedded wallet, gas sponsored. all five tiers are visible from the dashboard with a soft “locked” state for ones not yet earned.

solidity erc-721
04

gentle ui

soft blues and pastel orbs, large copy, no aggressive ranking. the marketing surface frames the product as a companion, not a coach. zero leaderboard.
